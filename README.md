# Información
### Para compilar el proyecto (*Dentro de la carpeta de public y functions))
##### 1. npm install

### Para compilar el proyecto:
##### 2. npm run dist (*Dentro de la carpeta de public)
¡
### Para correr el proyecto local:
##### 2. npm run dist (*Dentro de la carpeta de public)
##### 3. firebase serve (*Dentro de la carpeta root)

### Para subir el proyecto:
##### 3. firebase deploy
